[![Codecov](https://codecov.io/gh/<OWNER>/<REPO>/branch/main/graph/badge.svg)](https://codecov.io/gh/<OWNER>/<REPO>)


Opinionated starter repo for a vertical construction AI co-pilot. Includes a minimal Prisma MongoDB schema and a developer `docker-compose.yml` to run local dev services (MongoDB, Redis, Qdrant, MinIO).

Quick start

1. Copy `.env.example` to `.env` and set `DATABASE_URL` (MongoDB connection string) and any other secrets.
2. Start local dev services:

```bash
docker compose up -d
```

3. Install Prisma client (Node project):

```bash
npm install @prisma/client prisma --save-dev
npx prisma generate
```

4. Run the API or services in `api/` once scaffolded.
